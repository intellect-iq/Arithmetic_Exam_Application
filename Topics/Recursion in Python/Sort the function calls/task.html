Sort the function calls

Let's go back to our function for finding the factorial:

def recursive_factorial(n):
    # negative numbers have no factorials
    if n < 0:
        print("no solution")
    # base case
    elif n == 0:
        return 1
    # recursive case
    else:
        return n * recursive_factorial(n - 1)
